Portfolio Website Design

Overview

A modern dark-themed personal portfolio website built to showcase my profile, skills, and featured projects. The design emphasizes clarity, responsiveness, and accessibility across all devices.

Features

  • Responsive layout (desktop & mobile)
  • Clean navigation menu
  • Stylish dark UI with green accents
  • Project gallery with clickable detail pages

Tech Stack

  • HTML5 & CSS3
  • Google Fonts (Poppins)
  • Media Queries for responsiveness
  • Basic JavaScript for navigation behavior (optional)

What I Learned

This project helped me improve layout structuring with Flexbox, mobile-first design using media queries, and choosing good color contrasts for UI accessibility.
